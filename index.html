Museum RoboGuide

Project Description

Human Robot Interaction project is made with purpose of integrating robots in daily life, but to make it useful and productive. The project is made for museum visitors in order to increase interest among them and help to learn something new. Combination of Microsoft Kinect and Humanoid Robot Nao can help museum visitors with orientation within a facility, so people get useful information about exhibit in time.

Project Technologies and Hardware to be used

Microsoft Kinect - device which is able to recognize human and also enables users to control and interact with different device without using any remote controls, but through a natural user interface using gestures and spoken commands. Humanoid Robot Nao - robot prototype with a variety of hardware devices installed, such as sensors, audio input and output, cameras and etc, which make interaction with robot easier and attractive.

Problem Statement

The project will be based on the autonomous tour-guide/tutor robot, which will programmed to perform tours in the museums. Comparing with ordinary guides robots are able to readjust to different age groups in order to make the museum program interesting and understandable for every visitor starting from youths to the elderly part of the society. Moreover, the material presented by robots will not only be adjusted for different age groups, but will also be focused on explaining the “real” meaning of the exhibit. To make it more clear, the presentation performed by people is focused on simply giving general knowledge, history and the techniques used by the author, however the work of robots will be focused on delivering and explaining the significance of author’s work. Currently, people do not try to acquire the most significant point of the exhibits, thus the robot will be focused on making the audience engaged in the process.
